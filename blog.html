Simple Markdown-Based Blogging

Adding a new blog post is super easy! Just follow these steps:

Step 1: Create a Markdown File

Create a new file in blog/posts/ with the format: YYYY-MM-DD-title.md

Example: 2024-12-01-my-first-post.md

Step 2: Add Frontmatter

Start your markdown file with frontmatter (metadata):

---
title: "My First Blog Post"
date: "2024-12-01"
tags: ["data-science", "python"]
description: "A short description for SEO"
---

Step 3: Write Your Content

Write your blog post in markdown below the frontmatter. Use headings, lists, code blocks, and more!

Step 4: Commit & Push

Commit your new markdown file to GitHub, and it will automatically appear on your blog!
